# Ecommerce Analysis Skills

面向电商经营与竞品分析的可复用 AI Skills 仓库。

## Skills

### 生意参谋竞品 Dashboard

目录：[`shenyi-competitive-dashboard/`](shenyi-competitive-dashboard/)

用于审计淘宝/天猫生意参谋竞争模块导出的竞店、竞品 Excel 数据，规范化区间指标，并在确认业务口径后生成可验证的交互式 Dashboard。

主要能力：

- 审计 Excel 的 Sheet、字段、日期窗口、缺失值和重复业务键
- 规范化脱敏区间值，保留低/中/高估值口径
- 分析店铺、商品、流量、关键词和 SKU
- 以明确公式生成 GMV 估算，避免将推算结果表述为精确事实
- 验证 Dashboard 数据结构和关键交互

使用前请阅读该技能的 [`SKILL.md`](shenyi-competitive-dashboard/SKILL.md)。

## 重要说明

- 本仓库不包含任何真实客户数据、生意参谋导出文件或生成的业务 Dashboard。
- 生意参谋中的部分指标为平台脱敏区间，分析结果不应表述为精确事实。
- 当前仓库尚未添加开源许可证；除 GitHub 平台允许的浏览与 Fork 外，未授予额外复制、修改或分发许可。
